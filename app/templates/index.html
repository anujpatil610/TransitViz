About This Portal

TransitViz is a web application designed by Anuj Patil to facilitate the exploration and analysis of public transit systems. It leverages General Transit Feed Specification (GTFS) data to provide insights into transit routes, stop timings, and overall network efficiency. Users can explore static and dynamic transit data to better understand and improve urban mobility.

Features Overview

  • Interactive Maps: Visualize transit routes and stops on interactive maps.
  • Data-Rich Dashboards: Dive into detailed analytics on transit operations with customizable dashboards.
  • Real-Time Updates: (Planned Feature) Track buses and trains in real-time across the network.
